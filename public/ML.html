Overview
What is UWB?
Ultra-wideband is a short-range radio technology for precise indoor positioning. Signal quality depends on whether the path is clear (LOS) or blocked (NLOS) — and the difference drives ranging error.
Classification task
Binary classification — is an incoming UWB signal travelling via a clear line-of-sight path, or is it reflected and diffracted around obstacles? Getting this right is the first step toward reliable ranging.
Regression task
After identifying signal type, a regressor estimates the true distance — correcting for the ranging error that NLOS propagation introduces into raw measurements from the sensor hardware.
